# Project 1 — Loom to Bloom

Loom to Bloom is a one-page interactive website about Indian textiles and the designers who are reimagining them in contemporary fashion. The experience is designed as an editorial and immersive web piece rather than a commercial or product-based site. It begins with a cinematic collage and transitions into a five-panel interactive layout where each collection opens into its own visual world through hover-based interaction.

## Design Intent

### Core Idea
This project is a one-page interactive website about Indian textiles and selected designers who reinterpret them through contemporary fashion. The goal is to create a visually immersive and editorial digital experience that foregrounds textile richness, silhouette, mood, and storytelling.

### Mood
The project should feel warm, rich, editorial, cinematic, luxurious, romantic, heritage-driven, and immersive.

### Color Palette
- **Primary Background:** `#532128`
- **Text Color:** `#FFF39E`
- **Intro Tint 1:** `#532122` at 25% opacity
- **Intro Tint 2:** `#220001` at 40% opacity
- **Stage 2 Overlay:** `#532128` at 25% opacity
- **Panel Overlay:** `#330304` at 10% opacity

### Typography
- **Heading:** Cormorant Garamond, bold, large editorial scale
- **Subheading:** Neue Haas Grotesk, medium
- **Body:** Neue Haas Grotesk, regular
- **Panel Labels:** Neue Haas Grotesk, bold

### Layout Intent
The website moves through three visual states:
1. **Stage 1 — Intro Collage:** a full-screen cinematic opening that establishes mood and context
2. **Stage 2 — Five-Panel Selection Screen:** five equal full-height vertical panels that present the collections clearly
3. **Stage 3 — Hover / Expanded State:** one panel expands to dominate the screen while the others compress and recede

### Interaction Intent
The core interaction is hover-based. When the user hovers over a panel, that panel should become the dominant focus and reveal the following in a restrained sequence:
- brand name
- designer name
- collection name
- short supporting description

The interaction should feel like stepping deeper into a collection rather than triggering a standard gallery effect.

### Collections Included
1. **Tilla — The Vintage Project** — Aratrik Dev Varman  
2. **Injiri — Rasa** — Chinar Farooqui  
3. **Payal Khandwala — Release 13** — Payal Khandwala  
4. **Torani — Juloos** — Karan Torani  
5. **Anita Dongre — Crafts of India 2025** — Anita Dongre  

### Non-Negotiables
- The website must feel editorial, cinematic, and textile-rich
- The imagery must foreground garment, textile, and mood rather than portraiture alone
- Hover interaction must create a clear hierarchy shift
- The five collections must feel distinct but still belong to one cohesive world
- Typography must remain elegant and fashion-forward
- The final experience should prioritize art direction and storytelling over unnecessary complexity

### Constraints
- Single-page only
- No routing
- No backend functionality
- Interaction should stay visually immersive without becoming overly complex
- The focus should remain on visual concept, typography, imagery, and interaction design
